# AI Usage Disclosure

This document exists so nobody has to guess. Here's exactly which tools were
used, for what, and where they stand now.

## Tools Used, and Current Status

- **Claude (Anthropic)** — Primary, ongoing. Used throughout this project's
  documentation, debugging, and structural build-out. Actively in use.
- **Gemini (Google)** — Used generically for testing, and required for the
  Build with Gemini XPRIZE hackathon build (Duet: Solo), which specifically
  calls the Gemini API. Not a general-purpose tool of choice — used where
  the project or competition required it.
- **GitHub Copilot** — Used previously. No longer in use.
- **Z.ai** — Used previously. No longer in use.

## What AI Tools Were Used For

- **Documentation restructuring** — `duet.html` was reorganized into
  `duet_engine_architecture.html`, with an Ordered Engine Sequence added for
  clarity.
- **Bug fixing** — a Veil State refresh bug was identified and resolved with
  AI assistance.
- **Design system alignment** — the visual palette was brought in line with
  the project's brand design tokens.
- **Documentation accuracy** — conflicting tech-stack references were
  reconciled, and prior documentation that overclaimed project status
  (particularly around accessibility and funding positioning) was corrected.
- **Backend development** — the Duet: Solo hackathon build's Express backend
  (`server.js`) and its Gemini API integration were built with AI assistance.
- **General organization** — years of pre-existing original fiction and
  worldbuilding were structured into the documentation, lore, and design
  bible you see across this ecosystem.

## What AI Tools Were NOT Used For

- The characters, plot, philosophy, and worldview behind this project.
  Every story, character, and theme predates any AI tool's involvement —
  most by years, some by close to a decade.
- Creative direction. AI assistance here functions as an editor: organizing,
  structuring, and cross-referencing a world that already existed, not
  inventing one.

## Why This Document Exists

Because "AI was used somewhere in here" is a vague statement that invites
speculation, and speculation is worse than specifics. If you're evaluating
this project — as a collaborator, a contributor, a reviewer, or just someone
curious — this is the honest, itemized account. Ask if something here isn't
clear enough.
